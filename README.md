# Student API
A simple Student API built with Node.js and Express.
## Features
- Get all students
- Get a student by ID
- Add a new student
- Update a student
- Delete a student
## Technologies Used
- Node.js
- Express.js
## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mobolajioyeleye07-sudo/student-api.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   node server.js
   ```
The server runs on:
```
http://localhost:4000
```
